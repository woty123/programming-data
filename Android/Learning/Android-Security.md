# Android 平台安全、加密记录

## 1 Android RSA 加密模式和 JAVA 默认不一致

1. JDK 默认使用：`RSA/ECB/PKCS1Padding`。
2. Android 默认使用：`RSA/ECB/NoPadding`。

在开发中如果后台与客户端涉及到 RSA 加解密，需要注意统一好加密模式。

```java
Cipher cipher = Cipher.getInstance("RSA/ECB/PKCS1Padding");
```
